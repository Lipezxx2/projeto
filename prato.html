Guia HTML

As tags <h1 até h6 > são usadas para criar títulos em diferentes níveis de importância, sendo h1 o mais importante e h6 o menos importante.

A tag <p> é usada para escrever parágrafos de texto em uma página HTML, organizando o conteúdo de forma legível.

A tag <img> é usada para inserir imagens em uma página HTML. O atributo src indica o caminho da imagem e o alt fornece uma descrição caso a imagem não carregue.

A tag <a> é usada para criar links clicáveis em uma página HTML. O atributo href define o endereço para o qual o link vai direcionar.

As tags <ul>, <ol> e <li> são usadas para listas:

  • <ul> → lista não ordenada (bolinhas).
  • <ol> → lista ordenada (numerada).
  • <li> → cada item da lista, usado dentro de
      ou
        .

    Exemplo explicativo: "As tags <ul> e <ol> são usadas para criar listas em HTML: <ul> cria listas com marcadores e <ol> cria listas numeradas. Cada item da lista é colocado dentro da tag <li>."
